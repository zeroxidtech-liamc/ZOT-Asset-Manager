# ZOT-Asset-Manager
Gestión interna de assets de planta OT/IT para ZOT Chemical Solutions. Servicio ligero Node/Express con autenticación básica y panel de visualización. Incluye captura de credenciales Wi-Fi, inventario de equipos críticos (PLCs, HMIs, variadores) y preparado para integración con logs Modbus.
Servicio interno para visualizar y gestionar assets de planta.

## Uso rápido
```bash
npm install express express-session
node server.js
